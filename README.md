# React Quiz App

A simple quiz application built with React as part of my React learning journey.

## About

This project was created by following a GreatStack React tutorial to understand the fundamentals of React development. While building it, I learned how to manage state, handle user interactions, use refs, and implement conditional rendering.

## Features

* Multiple-choice quiz interface
* Instant answer validation
* Correct and incorrect answer highlighting
* Score tracking
* Final results screen
* Reset quiz functionality

## Technologies Used

* React
* JavaScript (ES6+)
* CSS3
* Vite

## Concepts Learned

* Functional Components
* `useState`
* Event Handling
* Conditional Rendering
* State Management
* Dynamic UI Updates
